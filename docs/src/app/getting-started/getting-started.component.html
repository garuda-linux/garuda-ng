What is GarudaNG?

GarudaNG is a component library for Angular built on top of PrimeNG for website-based projects of Garuda Linux.

Configuration

GarudaNG must be configured by providing it ideally in the ApplicationConfig of your angular application. This can be done using the provideGarudaNG function. If your application is already using PrimeNG, then this will replace your providePrimeNG call. All PrimeNG specific configuration can be put after the GarudaNG configuration.
